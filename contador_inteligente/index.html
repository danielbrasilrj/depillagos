Cenário B — Coworking Estético

Descartado

Profissionais pagam taxa fixa de uso do espaço. O salão fatura sobre a taxa, não sobre o serviço todo.

✗
Não faz sentido para a Depillagos Como a cota parte já é deduzida da NFS-e, o Simples já incide só sobre os 50% do salão. No coworking, a taxa de uso seria menor que os 50% atuais — o salão receberia menos receita. A economia de imposto não compensa a perda.

EXEMPLO: Manicure faz R$ 65 (pé e mão)

ModeloReceita DepillagosImposto (~10%)Lucro líquido
Atual (50/50) R$ 32,50 R$ 3,25 R$ 29,25
Coworking (taxa R$ 20) R$ 20,00 R$ 2,00 R$ 18,00

Mesmo pagando menos imposto, o salão perde R$ 11,25 por atendimento. Em ~1.000 serviços de manicure/mês, isso seria -R$ 11.250/mês de receita.

Quando faria sentido? Apenas se o CNAE de "locação de espaço" tivesse uma alíquota drasticamente menor no Simples. Na prática, para salões que já aplicam cota parte corretamente, o coworking é desvantagem.

Risco Fiscal na Prática — Araruama, R$ 100K/mês

O que joga a favor:

Cidade pequena Araruama (140K hab) não tem fiscalização local agressiva. Receita Federal regional foca em empresas > R$ 1M/mês, importadoras e setores de risco.
Faturamento R$ 100K "Baixo radar". Cruzamento automático existe, mas prioriza inconsistências grosseiras (faturamento vs. movimentação bancária, notas frias).
Setor de beleza Não é setor de alto risco. Salões não costumam ser alvos prioritários de fiscalização.

O que pode dar errado:

Mesmos sócios Se Empresa 1 e 2 têm Luciene como sócia, a Receita cruza automaticamente os CNPJs. Gatilho mais comum de fiscalização em holdings pequenas.
Valores inflados Cobrar R$ 15K/mês de "licenciamento de marca" sem registro INPI e sem contrato = fraude clara.
Empresa 2 fantasma Sem atividade real, sem funcionários, sem despesas próprias. Existe só pra receber dinheiro. Receita descarta.
Denúncia Ex-funcionário ou parceiro insatisfeito. Em cidade pequena, todo mundo sabe da vida de todo mundo.

8. NFS-e por cada MEI — Como resolver na prática

Para a Lei do Salão-Parceiro funcionar 100%, cada MEI precisa emitir nota fiscal da sua cota parte. Hoje isso não acontece de forma consistente. Aqui estão as opções:

Opção 1 — Recepção emite para todas (Procuração)

Mais prático

Cada MEI assina uma procuração simples autorizando a Depillagos a emitir NFS-e em nome dela. A recepção (Yasmin ou Simone) emite todas as notas de uma vez, semanalmente.

Rotina semanal (sábado, junto com o fechamento)

1
Trinks gera relatório de comissões da semana
Já existe — é o mesmo relatório usado pra pagar as parceiras.
2
Recepção acessa o Emissor Nacional de NFS-e
nfse.gov.br — portal gratuito do governo. Login com certificado digital ou Gov.br de cada MEI (configurado uma vez com procuração).
3
Emite uma NFS-e por MEI com o valor da cota parte
Ex: Jhennifer fez R$ 2.400 na semana → NFS-e de R$ 1.200 (50% dela). São ~12 notas, ~3min cada = ~40 minutos.

Opção 2 — Cada MEI emite pelo celular (App Gov.br)

Depende das profissionais

O app NFS-e Mobile (Gov.br) permite que cada MEI emita nota pelo celular. A recepção envia por WhatsApp: "Jhennifer, emite nota de R$ 1.200 pra Depillagos, CNPJ tal".

⚠
Problema real: Manicures em Araruama provavelmente não vão fazer isso de forma consistente. Esquecimento, dificuldade com app, falta de costume. Funciona no primeiro mês, depois para. Só vale como backup.

Opção 3 — Automação via API NFS-e Nacional

Futuro (Daniel pode construir)

O Emissor Nacional de NFS-e tem API REST que permite emissão automática. Importante: a API do Trinks emite NFS-e apenas da empresa — as notas das MEIs precisam ser emitidas via API NFS-e Nacional (nfse.gov.br).

1
Puxar valores de comissão do Trinks + Avec
Trinks API fornece os dados de atendimentos e valores. Avec complementa com comissões de manicure/cabeleireiras. O Trinks não emite NFS-e das MEIs.
2
Emitir NFS-e via API NFS-e Nacional (nfse.gov.br)
API REST gratuita do governo. Precisa de certificado digital de cada MEI ou procuração eletrônica via e-CAC. Projeto depillagos_notas_fiscais/ já trabalha com NFS-e e certificados — reaproveitar.
